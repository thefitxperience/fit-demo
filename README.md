# FIT Expo Demo - Body Composition Diet Simulator

A mobile-first web application that generates personalized diet plans based on real body composition data and user preferences.

## 🎯 Overview

This demo application showcases FIT's body composition diet planning capabilities. Users can input their personal information, body composition metrics, and dietary preferences to generate a customized diet plan with meal recommendations.

## ✨ Features

- **Multi-step Form Interface**: Intuitive 3-step wizard for data collection
  - Personal Details (name, gender, activity level, goals)
  - Body Composition (height, weight, muscle mass, BMR, etc.)
  - Medical Information (conditions and allergies)

- **Smart Calculations**: Auto-calculates daily calorie needs based on BMR and activity level

- **Real-time Preview**: Live preview panel showing generated diet plan

- **Personalized Plans**: Supports multiple dietary types (Omnivore, Vegan, Vegetarian, etc.) and plan styles (Regular, Intermittent Fasting)

- **Mobile-Optimized**: Landscape mode lock for optimal viewing experience on tablets

## 🛠️ Technology Stack

- **Frontend**: Pure HTML5, CSS3, and Vanilla JavaScript
- **Mobile**: Capacitor for Android APK generation
- **API Integration**: REST API for diet plan generation
- **Styling**: Modern gradient design with responsive layouts

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/andyayas/FIT-Expo-Demo.git
cd FIT-Expo-Demo
```

2. Install dependencies:
```bash
npm install
```

3. Open `index.html` in a web browser or serve it using a local server

### Building for Android

See [BUILD_APK.md](BUILD_APK.md) for detailed instructions on building the Android APK.

## 📱 Usage

1. **Personal Details**: Enter your name, gender, activity level, dietary preferences, and goals
2. **Body Composition**: Input your body metrics (optional fields will auto-calculate if BMR is provided)
3. **Medical Info**: Select any medical conditions or allergies
4. **Generate**: Click "Generate Plan" to create your personalized diet plan

## 🔧 Configuration

The application connects to the FIT API for:
- Fetching dietary types, activity levels, conditions, and allergies
- Generating personalized diet plans

API configuration is located in the `API_CONFIG` object in `index.html`.

## 🎨 Design Features

- **Gradient Theme**: Professional teal-to-navy gradient throughout
- **Responsive Layout**: Adapts from single-column to multi-column based on screen size
- **Smooth Animations**: Transitions between steps with zoom effects
- **Loading States**: Clear visual feedback during plan generation
- **Portrait Lock**: Ensures optimal tablet experience in landscape mode
